# Overview

The software I created is a console-based To-Do List Manager written in C++. It allows users to manage tasks by adding, deleting, updating, and marking them as complete or incomplete. The application uses object-oriented principles, handling task data with STL vectors, and features file I/O to save and load tasks from a file.

I wrote this software to practice my C++ skills, focusing on object-oriented programming and using STL for task management.
